## Comment utiliser ce repo

1. **Cheatsheets** : Chaque cours et box a son propre fichier `.md`
2. **Progression** : Cocher les `[ ]` -> `[x]` dans ce README au fur et a mesure
3. **Templates** : Utiliser `TEMPLATE_COURS.md`, `TEMPLATE_BOX.md` ou `TEMPLATE_PROLAB.md` pour ajouter du contenu
4. **Commit regulier** : Commiter apres chaque cours/box termine pour garder un historique
