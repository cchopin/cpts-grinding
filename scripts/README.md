# Scripts

Scripts d'automatisation developpes au fil des boxes et des cours.

## Usage

Ajouter ici les scripts d'automatisation, d'enumeration, ou tout outil custom cree pendant le grinding.

## Exemples d'idees

- Script d'enum initial (nmap + gobuster + whatweb en une commande)
- Script de generation de reverse shell adapte
- Script d'extraction de credentials depuis des fichiers
- Wrapper autour de ffuf avec les options par defaut
